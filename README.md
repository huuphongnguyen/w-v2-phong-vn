## Running Locally

```bash
$ git clone https://github.com/huuphongnguyen/phong.vn.git
$ cd phong.vn
$ npm install
$ yarn dev
```

## Built Using

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [Tailwind CSS](https://tailwindcss.com/)
